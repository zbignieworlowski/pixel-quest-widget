Setup Instructions

  1. Go to StreamElements ‚Üí My Overlays ‚Üí New Overlay
  2. Name it "Pixel Quest" and click Create
  3. Click Add Widget ‚Üí Static/Custom ‚Üí Custom Widget
  4. Click on the widget ‚Üí Open Editor
  5. Copy each code section into the matching tab (HTML, CSS, JS, Fields, Data)
  6. Click Save
  7. Use the Settings panel to customize colors, icons, and options
